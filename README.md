# Resume Genie 🚀

Resume Genie is an AI-powered career assistant built using Streamlit, LangChain, and Google's Gemini 2.5 Flash model.

## Features

### 📄 Resume Reviewer

Analyzes resumes and provides detailed feedback, strengths, weaknesses, and improvement suggestions.

### 🎯 ATS Resume Matcher

Compares resumes against job descriptions and generates an ATS compatibility score along with skill gap analysis.

### ✉️ Cover Letter Generator

Generates personalized ATS-friendly cover letters based on the uploaded resume and target job description.

### 🤖 AI Career Coach

Provides career guidance, interview preparation tips, skill recommendations, and job search strategies based on the candidate's resume.

## Tech Stack

* Python
* Streamlit
* LangChain
* Google Gemini 2.5 Flash
* PyPDFLoader
* Git & GitHub
